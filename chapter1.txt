To initialize the git -> git init
To add the file in staging area -> git add <filename.txt>
To commit the files -> git commit -m "savepoint message"

To check the difference between last and current version -> git diff <filename.txt>
To rollback to last version -> git checkout <filename.txt>

To create git remote -> git remote add <remote_name> <url_of_created_repositories>
To push the repositoris from local to remote -> git push -u <remote_name> <branch_name>

To remove files from staging -> git rm --cached -r .

Useful repos for beginers: https://github.com/MunGell/awesome-for-beginners?tab=readme-ov-file
                           https://github.com/clupasq/word-mastermind

To create a branch -> git branch <branch_name>
To display all the branch created -> git branch 
The asterisk symbol shows the current branch we are working
To switch one to another branch -> git checkout <branch_name_to_switch>
